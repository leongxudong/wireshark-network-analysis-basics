# Wireshark Network Analysis Basics

A lightweight portfolio project documenting basic packet analysis workflows using Wireshark.

## Project Purpose

The purpose of this repository is to demonstrate network fundamentals through practical packet analysis examples.

## Scope

This project covers:

- Useful Wireshark display filters
- TCP three-way handshake review
- DNS query and response analysis
- HTTP visibility and TLS limitations
- Screenshot placeholders for packet analysis examples

## Repository Structure

```text
wireshark-network-analysis-basics/

    └── README.md
```

## Topics Covered

| Topic | Purpose |
|---|---|
| Display filters | Quickly isolate relevant traffic |
| TCP handshake | Understand session establishment |
| DNS analysis | Review name resolution behaviour |
| HTTP observation | Understand cleartext web request visibility |
| TLS observation | Understand what is and is not visible in encrypted traffic |

## Portfolio Value

This project demonstrates:

- Protocol literacy
- Network troubleshooting fundamentals
- Packet capture interpretation
- Technical evidence capture
- Ability to communicate technical observations clearly

## Disclaimer

This repository contains learning notes and sanitized examples only. Packet captures should be generated from lab traffic or authorised environments.
